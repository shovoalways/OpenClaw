## 🎯 Setup API Key
Openrouter > Credits > API Key > Create 

Hostinger > Docker Manager > Projects > Manage > Environment > ADD new Environment > OPENROUTER_API_KEY > Save and Deply

```bash
I have added the OPENROUTER_API_KEY as an environment variable

I want to optimize my model configuration to reduce costs. Please make these changes:

Set my primary model for general tasks to use openrouter/minimax/minimax-m2.5
Set fallbacks to: openrouter/anthropic/claude-haiku-4-5, then openrouter/google/gemini-2.5-flash
```



## 🎯 Heart bit checks 
```bash
Please configure my heartbeat settings for cost optimization:
1. Set the heartbeat interval to every 60 minutes
2. Set heartbeat model to openrouter/google/gemini-2.5-flash-lite (the cheapest available 
3. Set heartbeat target to "last"
```
